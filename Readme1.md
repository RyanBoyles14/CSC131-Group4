## Usage
```
mvn clean install
java -jar target/Group4-0.0.1-SNAPSHOT.jar [options] <url>
```

### Options

    -h, --help
    -c, --coupling
    -H, --halstead
    -I, --inheritanceDepth
    -C, --commits
    -t, --timeComplexity
